# My Dify

一个现代化的文档管理和预览系统，基于 React + TypeScript + Vite 构建。

## 功能特点

- 📄 多格式文档支持

  - PDF 文件预览
  - Word/Excel 文档预览
  - 图片预览

- 📱 响应式布局

  - 列表视图
  - 网格视图
  - 卡片视图

- 🎯 智能管理

  - 文档分组
  - 标签管理
  - 拖拽排序
  - 批量操作

- 📊 数据统计
  - 检索次数统计
  - 命中率分析
  - 可视化展示

## 快速开始

1. 克隆项目

```bash
git clone git@github.com:hakityc/my-dify.git
cd my-dify
```

2. 安装依赖

```bash
pnpm install
```

3. 启动开发服务器

```bash
pnpm run dev
```

4. 构建生产版本

```bash
pnpm run build
```

## 技术栈

- React 18
- TypeScript
- Vite
- Tailwind CSS
- Shadcn/ui
- ECharts

## 开发环境要求

- Node.js 16+
- npm 7+
